# PhotoLand E-commerce

Welcome to **PhotoLand** - A modern e-commerce platform for photography enthusiasts.

## 📋 About PhotoLand

PhotoLand is a full-featured e-commerce platform built with modern web technologies, designed to provide a seamless shopping experience for photography equipment, accessories, and services.

## 🛠️ Tech Stack

- **Frontend Framework**: React with Vite
- **Build Tool**: Vite
- **Languages**: 
  - JavaScript (96.5%)
  - CSS (3.1%)
  - HTML (0.4%)
- **Development**: React plugins for Fast Refresh (Babel or SWC)

## 🚀 Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone https://github.com/Basharkatrib/PhotoLand-ecommerce.git
cd PhotoLand-ecommerce
```

2. Install dependencies
```bash
npm install
```

3. Start the development server
```bash
npm run dev
```

The application will be available at `http://localhost:5173`
